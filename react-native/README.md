# Le Tre Gemme Mobile

Struttura iniziale React Native pensata per consumare le API FastAPI del backend.

## Setup futuro suggerito

Questa cartella e uno scaffolding iniziale. Per inizializzare davvero il progetto si puo usare Expo:

```powershell
npx create-expo-app frontend-react-native --template
```

Poi mantenere la struttura `src/` proposta qui.

## Variabili ambiente

API base locale:

```text
EXPO_PUBLIC_API_BASE_URL=http://localhost:8000/api
```
